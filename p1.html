Project #1 Lost in Translation

Description

A silly cat made of rectangles. :D

Design Process

I made a lil cat in google spreadsheets using a grid system to create pixel art. I then coded the cat in p5, using the grid system as a guide to line up each created rectangle.

Reflections

It was rough figuring out the changes in the x y axis and how to line up each rectangle but the grid helped speed up the process. Once I figured out the math, evrything began to flow. It just took some time to figure out the RGB system for colors!
